## ✨ Features / Tech stack

- 📦 Docker
- 📦 Docker-compose
- 🛡 mosquitto mqtt broker
- 🛡 Paho mqtt client
- 🌍 Python
-   Timescaledb

## 📦 Install

```bash
# clone the project

cd <project-folder>
docker-compose build
```

## 🔨 Usage

```bash
docker-compose up
# view README in agrobot_api directory
# view test_scripts/localhost_tests
# view topic shema in ControllerClientTopicShema.txt
```

## ⌨️ Development

```bash
export ENV=development
docker-compose up
```
